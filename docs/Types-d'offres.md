# Types d'offres

L'outil supporte les 4 types d'offres éligibles au CPF sur Mon Compte Formation.

## Formation certifiante

Code certification : **RNCP** ou **RS** (Répertoire Spécifique)

- Recherche automatique de la fiche RNCP/RS via l'API France Compétences
- Import des blocs de compétences
- Vérification de l'habilitation à former

## Permis de conduire (CPF206+)

Codes CPF selon la catégorie :
- CPF206 : Permis B
- CPF210 : Permis C
- CPF214 : Permis D
- CPF219 : Permis A1
- CPF220 : Permis A2
