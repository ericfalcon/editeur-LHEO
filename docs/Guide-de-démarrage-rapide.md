# Guide de démarrage rapide

## Prérequis

- Python 3.8+ (pour le serveur proxy local)
- Un navigateur moderne (Chrome, Firefox, Edge)

## Installation locale (recommandée)

1. Télécharger les fichiers depuis [GitHub](https://github.com/ericfalcon/editeur-LHEO)
2. Lancer `lancer-editeur.bat` (Windows) ou `python lancer-editeur.py`
3. L'outil s'ouvre sur `http://localhost:8765`

## Utilisation en ligne

Accessible directement sur [GitHub Pages](https://ericfalcon.github.io/editeur-LHEO/editeur-lheo-edof.html).

> ⚠️ La vérification SIRET et la recherche RNCP nécessitent le serveur proxy local.

## Créer votre première fiche

1. Cliquer **+ Nouvelle fiche**
2. Renseigner le SIRET de votre organisme
3. Choisir le **type d'offre** (formation certifiante, VAE, bilan, permis)
4. Remplir les sections Certification, Description, Actions, Sessions
5. Cliquer **Exporter tout** pour générer le XML EDOF
