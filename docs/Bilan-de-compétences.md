# Bilan de compétences

## Cadre légal (art. L6313-4 et R6313-4 du Code du travail)

### Durée
- **Maximum 24 heures** (obligatoire)
- Réparties sur plusieurs semaines

### 3 phases obligatoires

1. **Phase préliminaire** : définir les besoins, présenter la démarche
2. **Phase d'investigation** : analyser les motivations, identifier les compétences, explorer les pistes
3. **Phase de conclusion** : formaliser les résultats, définir le projet professionnel

### Document de synthèse
Remis exclusivement au bénéficiaire — document confidentiel.

## Spécificités EDOF

- Code CPF : **CPF202** (fixe)
- Certifiante : **Non**
- Objectif général : **9** (Mobilisation, aide au projet professionnel)
- Parcours de formation : **6** (Bilan de compétences)
- Qualiopi mention **bilan de compétences** obligatoire

## Calcul des frais

Montant HT = prix horaire HT × heures  
Montant TTC = HT × (1 + TVA%)

> Pour les bilans, la TVA est généralement à 0% (exonéré).
